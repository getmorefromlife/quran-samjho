# قرآن سمجھو — Quran Samjho

**Read. Understand. Reflect.**

Quran Samjho is a community service initiative by **Syed Imon Rizvi (Mercer and Mills)**, built to give everyone — regardless of background — a quiet, distraction-free space to read and understand the Holy Quran.

> *"Reading with understanding is an act of reflection. It opens the heart, quiets the mind, and draws us closer to what matters most."*

---

## Mission

Too often the Quran is recited without comprehension. This project is a humble effort to bridge that gap — offering the Arabic text alongside faithful Urdu and English translations in a calm, Zen-inspired interface. No ads. No clutter. Just the Book and its meaning.

Mercer and Mills is committed to **interfaith dialogue** and **humanitarian work**, because peace begins when we seek to understand one another.

---

## Features

| Feature | Description |
|---|---|
| **Three Reading Modes** | Monolingual (Arabic only), Single Translation, Comparative (3 side-by-side) |
| **Three Translations** | English (Qarai), Urdu (Jawadi), Urdu (Najafi) |
| **Complete Quran** | All 114 surahs, all 6,236 verses |
| **Zen UI** | Warm cream light mode, slate-olive dark mode, no visual clutter |
| **Font Controls** | 12 Arabic fonts, 2 Urdu fonts, 5 English fonts, adjustable size & spacing |
| **Verse Navigation** | Range slider + direct verse number input + search |
| **Auto-Scroll** | Play/pause with speed presets (3s–20s) for hands-free reading |
| **Bookmarks** | Save verses, continue from where you left off |
| **Responsive** | Works on HD, desktop, tablet, and mobile |

## Getting Started

```bash
git clone https://github.com/getmorefromlife/quran-samjho.git
cd quran-samjho
npm install
npm run dev
```

Open [http://localhost:3000/quran-samjho](http://localhost:3000/quran-samjho) (note the base path).

### Build for Production

```bash
npm run build
```

Static files are exported to the `out/` directory, ready to deploy to any static host.
